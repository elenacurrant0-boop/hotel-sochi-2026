## Что это за проект

**Финансовая модель отеля Aqva SPA Resort 4★ (Сочи) на 2026 год**

React + TypeScript + Vite SPA, деплой на Vercel. Защищённое приложение с паролями для разных ролей.

**Объект:** Aqva SPA Resort, Лоо/Сочи. Владелец — Илона. ~100 номеров, SPA, медцентр, бассейн с морской водой.

## Ключевые структуры данных в App.tsx

### Пакеты (7 штук)
`BB` / `HB` / `FB` / `ULTRA` / `SPA` / `MED` / `PROMO`

### Ценовые периоды `PRICE_PERIODS[]`
Массив объектов `{ name, dates, ... }`. Индекс 9 (P9) — период "01.11–28.12", он же используется для низкого сезона января/февраля с `displayDates`.

### Месяцы `MONTHS[]`
Каждый месяц имеет `distribution[]` с полями `pIdx`, `sKey`, `days`, опционально `displayDates`.

### Конкуренты (в exec-report)
11 объектов в регионах Лоо, Лазаревское, Адлер/Сириус. Только локальные Сочи-конкуренты — КМВ не включать.

### Сезонные продукты
Хранятся в `localStorage` под ключом `sochi_seasonal_products`. Флаг `inReport: true` — продукт попадает в Пояснительную записку.
